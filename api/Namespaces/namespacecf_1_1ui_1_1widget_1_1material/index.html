    enum ScrollbarState¶

    Enumerator Value Description
    Idle Idle state - 40% opacity, 12dp width
    Hovered Hovered state - 100% opacity, 16dp width
    Dragged Dragged/Pressed state - 100% opacity, 16dp width, state layer overlay

    Scrollbar interaction states for Material Design 3.

    Since: 0.1

    Defines the various visual states a scrollbar can be in following Material Design 3 state layer specifications.

    enum SeparatorMode¶

    Enumerator Value Description
    FullBleed Spans entire width/height.
    Inset 16dp margin on both sides
    MiddleInset 16dp margin on leading/start side

    Separator spacing mode.

    Since: N/A

    Defines the inset/spacing behavior of the separator line. Full-bleed spans the entire width/height, Inset adds margins on both sides, and Middle-inset adds margin on one side.
