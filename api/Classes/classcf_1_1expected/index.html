    Detailed Description¶

    template <typenameT ,
typenameE >
class cf::expected;

    A wrapper that contains either a value or an error.

    Template Parameters:

    • T Value type.
    • E Error type.

    Provides type-safe error handling without exceptions. The object either contains a value of type T or an error of type E.
