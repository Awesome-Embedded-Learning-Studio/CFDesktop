    Detailed Description¶

    struct cf::desktop::metainfo::Build;

    Build metadata (compile-time determined).

    Exceptions:

    • None

    Since: N/A

    Note: None

    Warning: None

    Provides compiler, build type, platform, and timestamp information. All methods are constexpr for zero runtime overhead.
