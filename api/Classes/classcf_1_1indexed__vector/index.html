    Detailed Description¶

    template <typenameT ,
IndexingMode DefaultMode =IndexingMode::Listed>
class cf::indexed_vector;

    A cursor-enhanced vector with configurable navigation modes.

    Template Parameters:

    • T Element type.
    • DefaultMode Compile-time default indexing mode (Listed if omitted).

    Wraps a std::vector<T> and adds an embedded cursor that tracks the "current" element. Three indexing modes control how [next()](classcf_1_1indexed__vector.md#function-next) and [prev()](classcf_1_1indexed__vector.md#function-prev) behave:

    • [IndexingMode::Fixed](../Namespaces/namespacecf.md#enumvalue-fixed): cursor is locked; [next()](classcf_1_1indexed__vector.md#function-next)/prev()are no-ops.
    • [IndexingMode::Listed](../Namespaces/namespacecf.md#enumvalue-listed): linear traversal; throws std::out_of_range on bounds.
    • [IndexingMode::Recycled](../Namespaces/namespacecf.md#enumvalue-recycled): circular traversal; wraps around at both ends.

    The default mode can be set at compile time via the DefaultMode template parameter, and changed at runtime with [set_mode()](classcf_1_1indexed__vector.md#function-set-mode).

    cf::indexed_vector<int> playlist{1, 2, 3, 4, 5};
playlist.set_cursor(2);   // cursor -> 3
playlist.next();           // cursor -> 4

// Listed mode: next() at end throws
playlist.set_mode(cf::IndexingMode::Recycled);
playlist.set_cursor(4);
playlist.next();           // wraps to cursor 0

    function next¶

    inline void next()

    Advances the cursor forward.

    Exceptions:

    • std::out_of_range (Listed) cursor at end or vector empty; (Recycled) vector empty.

    Since: 0.1

    Behavior depends on the current mode:

    • Fixed: No-op.
    • Listed: Increments cursor; throws if already at end.
    • Recycled: Increments cursor; wraps to 0 at end.

    function prev¶

    inline void prev()

    Moves the cursor backward.

    Exceptions:

    • std::out_of_range (Listed) cursor at start or vector empty; (Recycled) vector empty.

    Since: 0.1

    Behavior depends on the current mode:

    • Fixed: No-op.
    • Listed: Decrements cursor; throws if already at start.
    • Recycled: Decrements cursor; wraps to end at start.
